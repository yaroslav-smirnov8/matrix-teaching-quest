## Screenshot Guidelines

### Dimensions
- **Recommended**: 1200x800px or 1920x1080px
- **Minimum**: 800x600px
- **Maximum**: 2560x1440px

### Format
- **Format**: PNG or JPG
- **Quality**: High quality (80%+ for JPG)
- **File size**: Under 500KB per image (optimize if needed)

### Content
- Show actual application screens
- Include relevant UI elements
- Highlight key features
- Use consistent styling

### Optimization

Use tools like:
- **TinyPNG**: https://tinypng.com/
- **Squoosh**: https://squoosh.app/
- **ImageOptim**: https://imageoptim.com/

### Naming Convention

Use lowercase with hyphens:
- ✅ `loading-scene.png`
- ✅ `admin-dashboard.png`
- ❌ `Loading Scene.png`
- ❌ `admin_dashboard.png`
